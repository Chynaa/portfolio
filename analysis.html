Summary of Findings

Incorporation of Findings

After doing all my research and user testing, I have found that I would like to completely overhaul my entire site and redesign it from the ground up. This is the first website I have ever created so before I jump right into overhauling the site I want to work on improving my HTML and CSS skills. I was very unconfident in them when first designing my site and found that I encountered several issues. Some of these issues were never resolved because I was not even sure what to look for in a possible solution. I believe that if I work on these skills then come back to update my site, things will go much more smoothly this time around.

Outside of this major design overhaul, I need to add a link to my linkedin profile and resume. I would likely do this using icons symbolizing those two things on my About Me page. When doing competitor analysis, I found that every competitor had a very detailed description of each project and hey made sure to not include just the result. I did not realize this immediately but as someone interested in user research and testing as a career, the design research details are the most important part of each portfolio project. To implement this I would like to take advantage of the single page scrolling style that looks like you are navigating between multiple pages. Finally, I believe adding a navigation bar to the top of my site will be of high value to users. For now, if they are not already on the home page they must return the home page to access a different page of the site.

Even though I want to wait to make most of my other changes, I did make one small change to the color of the center background bar on my site. Several people told me that the black text on the black transparent background was too difficult to read. After trying a few alternatives, I found that changing it to a grey transparent color was better for reading site content.
